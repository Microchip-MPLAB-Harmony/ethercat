MPLAB Harmony Configurations

The following Project Graph diagram shows the Harmony components included for the EtherCAT demonstration application. 

 

 

 

EtherCAT configuration option - 

 

EtherCAT LAN9252 driver configuration w.r.t External Interrupt PIN configuration , SPI CHIP select PIN configuration and also EtherCAT priority level setting. 

 

  • "EtherCAT Interrupt Priority level" - This configuration allows to disable interrupts more specifically. The default value is 4 and maximum configured value is 7. These values refers to the priority values.

- For ARMv7-M (Cortex-M3/M4/M4F) provides additionally the BASEPRI special register, which allows to disable interrupts more selectively. Specifically this configuration parameter disable interrupts only with urgency lower than a certain level ( i,e priority 4,5,6 and 7 configured interrupts will be disabled) and leave the higher-urgency interrupts( I,e, priority 0, 1, 2,3 configured interrupts) not disabled at all. 

 

- For MIPS devices, which also allows to disable interrupts more selectively. IPL is [0 to 7] with 7 being the highest level. If "EtherCAT Interrupt Priority level" set to 4, for example then only interrupts with IPL > 4 will be active, the rest will be masked. 

 

  • LAN9252 SYSTEM interrupt PIN configuration -

There are three different Interrupt external events are recorded for LAN9252 device. Those are IRQ, SYNC0 and SYNC1 events. With respect to the interrupts, the MHC provides EIC/GPIO/PIO pin selection for External event registration and event handler processing.

  • LAN9252 MHC configuration also supports SPI chip select PIN selection. This PIN can be PIO/PORT/GPIO according to the device.
