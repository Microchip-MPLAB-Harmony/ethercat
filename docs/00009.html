EVB_Lan925x_application

This section describes Microchip's EtherCAT demonstration projects, including hardware compatibility. It also provides the information about how to configure and run the demonstrations on different hardware boards. 

 

EtherCAT version v3.0.0 releases - 

\ethercat\apps\ethercat_counter_foe_app\firmware\production\sam_d51_lan9252_counter_foe_v3_0.hex 

\ethercat\apps\ethercat_counter_foe_app\firmware\production\sam_d51_lan9252_counter_foe_v3_0.bin 

The above files will be used for "D51 LAN925x EVB" board validation. User can follow the "Running the application" part of the document .
