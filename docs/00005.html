**Linker Support** 

- ATSAMD51 device supports bank switching and EtherCAT use this feature for the Firmware upgrade to demonstrate FoE feature . The range of BANK-B is 0x40000 to 0x7ffff , this BANK-B is used to store the new EtherCAT firmware. 

-ethercat_foe.ld file is added manually to the demo to restrict the **ROM_LENGTH** to 0x40000. 

 

**NOTES** 

As the Fuse settings are disabled for the released demo to demonstrate FoE , we can observe this following **warning** 

**Configuration memory will not be programmed because no configuration bits settings have been defined in your code. To program configuration memory, either define the settings in your code or use the Program Configuration Bits button on the configuration memory window.** 

- Please ignore this warning
